# Changelog

## 1.0.0 - 2026-05-14

Initial stable release of Pi Resource Manager.

### Added

- Manage Pi packages, global extensions, and skills from an interactive Pi command.
- Quarantine-first safety model for local extensions and skills.
- Package update/remove actions through Pi package management.
- Trusted skill update support through lock metadata and git-managed skill repositories.
- Scrolling resource list for smaller terminals.
- Inline resource detail panel with Update, Delete, Read, Locate, and Back actions.
- GitHub install instructions, MIT license/contribution notes, and package/local row documentation.
